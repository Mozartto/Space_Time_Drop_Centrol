> **本项目是在校期间和室友一起给苏州一家新能源公司做的一个数据中控台，耗时20天左右。**
> 
> **具体功能是在后台自动统计子门店系统上传的销售数据，并在前端自动绘制数据曲线、图表展示在中控大屏上。**
> 
> **项目名字叫时空水滴（老板起的名字），central_control简写为Centrol，代表总部的数据中控台部分，sub仓库中还有各个子门店的系统**
> 
> **项目采用的是比较常规的Springboot+Vue架构，前后端均为自己编写**
>
